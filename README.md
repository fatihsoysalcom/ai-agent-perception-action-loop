# AI Agent Perception Action Loop

This Python script demonstrates a basic AI agent's Perceive-Decide-Act loop. It simulates an agent interacting with a simple environment, making decisions based on perceived states and performing actions.

## Language

`python`

## How to Run

Save the code as `ai_agent.py` and run it from your terminal using `python ai_agent.py`.
